# 微信自动点击工具

## 功能介绍

这是一个用于自动检测和点击微信"进入微信"按钮的工具，主要功能包括：

- 检测微信进程是否正在运行
- 将微信窗口自动置顶并激活
- 自动识别并点击绿底白字的"进入微信"按钮
- 提供详细的日志输出，便于排查问题

## 适用场景

- 微信登录后需要手动点击"进入微信"按钮的情况
- 自动化脚本中需要处理微信登录后的操作
- 批量处理微信登录时的重复性点击操作

## 安装步骤

### 1. 安装Python

确保你的系统已安装Python 3.6或更高版本。

### 2. 安装依赖库

使用pip安装所需的依赖库：

```bash
pip install pyautogui Pillow numpy pywin32
```

### 3. 下载代码

将`wechat_clicker.py`文件保存到你的电脑上。

## 使用方法

1. 确保微信已启动并显示"进入微信"按钮
2. 运行脚本：

```bash
python wechat_clicker.py
```

3. 脚本会自动检测微信进程，将窗口置顶，然后查找并点击"进入微信"按钮

## 工作原理

1. **进程检测**：使用`psutil`库检测微信进程是否运行
2. **窗口置顶**：使用`win32gui`库将微信窗口置顶并激活
3. **按钮识别**：通过屏幕截图和图像分析识别绿底白字的"进入微信"按钮
4. **自动点击**：使用`pyautogui`库模拟鼠标点击操作

## 注意事项

- 确保微信窗口在屏幕上可见
- 确保"进入微信"按钮为绿底白字样式
- 运行脚本时，请勿移动鼠标或遮挡微信窗口
- 如遇权限问题，请以管理员身份运行脚本

## 故障排除

- **找不到微信窗口**：请确保微信已启动且正在运行
- **找不到"进入微信"按钮**：请确保按钮在屏幕上可见，且为绿底白字样式
- **点击无效**：请检查屏幕分辨率和缩放设置，可能需要调整代码中的按钮识别参数

## 依赖库说明

- `pyautogui`：用于屏幕截图和模拟鼠标点击
- `Pillow`：用于图像处理
- `numpy`：用于图像数据分析
- `subprocess`：用于检测进程状态（Python标准库，无需单独安装）
- `pywin32`：用于Windows窗口操作
